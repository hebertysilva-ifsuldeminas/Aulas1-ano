Dominando Formulários

O Guia Completo com Bootstrap 5

Objetivo da Aula

Capacitar os alunos a construir formulários modernos, responsivos e visualmente atraentes com Bootstrap, aplicando as melhores práticas de estrutura, layout e validação.

A Base dos Formulários

A classe mais importante que você vai usar é a .form-control.

O que ela faz?

Aplica width: 100%, padding, bordas e estilos de foco a qualquer <input>, <textarea> e <select>.

Exemplo: O Essencial .form-control

Estruturando Elementos

Além do .form-control, existem classes para cada tipo de campo.

Selects

Use .form-select para estilizar caixas de seleção.

Checks & Radios

Use .form-check como um container para o input e o label.

Switches

Adicione .form-switch a um .form-check para um visual de interruptor.

Exemplo: Selects, Checks e Switches

Layouts de Formulário

Organize seus campos com o Grid System e Input Groups.

Grid

Use .row e .col-* para posicionar campos lado a lado de forma responsiva.

Input Groups

Adicione prefixos, sufixos ou botões diretamente colados aos seus inputs.

Floating Labels

Crie um efeito moderno onde o label "flutua" para cima do campo.

Validação de Formulários

Fornecer feedback ao usuário é crucial. O Bootstrap estiliza a validação nativa do navegador.

Como Ativar

Adicione a classe .needs-validation e o atributo novalidate ao seu <form>.

Adicione as divs .valid-feedback e .invalid-feedback após os campos.

Requer um pequeno script JavaScript para funcionar!

<!-- Select -->
<select class="form-select">

  <option value="1">Opção 1</option>
</select>

<!-- Checkbox -->
<div class="form-check">
  <input class="form-check-input" type="checkbox" id="check1">
  <label class="form-check-label" for="check1">Lembrar de mim</label>
</div>

<div class="form-check">

  <label class="form-check-label" for="radio1">Opção Rádio</label>
</div>

<!-- Switch -->
<div class="form-check form-switch">
  <input class="form-check-input" type="checkbox" role="switch" id="switch1">

</div>

Código: Validação
<!-- Adicione .needs-validation e novalidate ao form -->
<form class="row g-3 needs-validation" novalidate>
  <div class="col-md-6">
    <label for="validationCustom01" class="form-label">Primeiro nome</label>
    <input type="text" class="form-control" id="validationCustom01" required>
    <div class="valid-feedback">Tudo certo!</div>
    <div class="invalid-feedback">Por favor, insira um nome.</div>
  </div>
  <div class="col-md-6">
    <label for="validationCustom02" class="form-label">Sobrenome</label>
    <input type="text" class="form-control" id="validationCustom02" required>
    <div class="invalid-feedback">Por favor, insira um sobrenome.</div>
  </div>
  <div class="col-12">
    <button class="btn btn-primary" type="submit">Enviar</button>
  </div>
</form>
// SCRIPT NECESSÁRIO NO FINAL DA PÁGINA
(function () {
  'use strict'
  var forms = document.querySelectorAll('.needs-validation')
  Array.prototype.slice.call(forms)
    .forEach(function (form) {
      form.addEventListener('submit', function (event) {
        if (!form.checkValidity()) {
          event.preventDefault()
          event.stopPropagation()
        }
        form.classList.add('was-validated')
      }, false)
    })
})()
