Por que a Navbar é Essencial?

A navbar é o principal ponto de orientação do usuário em um site.

Navegação

Funciona como um mapa, permitindo que os usuários encontrem facilmente as seções mais importantes do site.

Branding

É o local ideal para exibir o logo da sua marca, reforçando a identidade visual em todas as páginas.

Acesso Rápido

Pode conter atalhos para ações importantes, como busca, login ou acesso ao carrinho de compras.

Anatomia de uma Navbar

É um cabeçalho de navegação que usa um conjunto específico de classes para funcionar.

Classes Fundamentais

.navbar: A classe principal que "ativa" o componente.

.navbar-expand-*: Ponto chave da responsividade. Define em qual *breakpoint* (sm, md, lg, xl) a navbar passará da versão "hambúrguer" para a expandida.

Componentes Internos

.navbar-brand para o logo ou nome do site.

.navbar-nav para a lista de links de navegação (<ul>).

.navbar-toggler para o botão que controla a exibição em telas pequenas.

.navbar-collapse para o container que agrupa e esconde os links.

Posicionamento da Navbar

Controle onde a navbar fica na página com classes utilitárias.

Padrão

A navbar fica no fluxo normal do documento e rola junto com o conteúdo.

Fixo no Topo (.fixed-top)

A navbar fica fixa no topo da janela, mesmo quando a página é rolada.

"Grudenta" (.sticky-top)

Começa no fluxo normal e "gruda" no topo quando você rola a página até ela.

Resumo: Checklist da Navbar Perfeita

Estrutura

Use .navbar e .navbar-expand-lg como base.

Responsividade

Combine .navbar-toggler e .navbar-collapse para o menu "hambúrguer" funcionar.

Customização

Use classes bg-* para cores e .fixed-top para posicionamento.

Código: Navbar Básica
<nav class="navbar navbar-expand-lg bg-body-tertiary">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Navbar</a>
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
      <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarNav">
      <ul class="navbar-nav">
        <li class="nav-item">
          <a class="nav-link active" aria-current="page" href="#">Home</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Features</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Pricing</a>
        </li>
      </ul>
    </div>
  </div>
</nav>

Código: Alinhamento com .navbar-nav
<nav class="navbar navbar-expand-lg bg-body-tertiary">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Alinhamento</a>
    <div class="collapse navbar-collapse">
      <!-- A classe "me-auto" (margin-end: auto) empurra todo o resto para a direita -->
      <ul class="navbar-nav me-auto mb-2 mb-lg-0">
        <li class="nav-item"><a class="nav-link" href="#">Home</a></li>
        <li class="nav-item"><a class="nav-link" href="#">Sobre</a></li>
      </ul>
      <!-- Estes itens ficarão na direita -->
      <span class="navbar-text">
        Itens à direita
      </span>
    </div>
  </div>
</nav>

<!-- Dica: Use "ms-auto" (margin-start: auto) para alinhar à direita -->
<!-- Use "mx-auto" (margin-x: auto) para centralizar -->
