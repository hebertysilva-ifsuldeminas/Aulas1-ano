Dominando Modais

O Guia Completo com Bootstrap 5

O que é um Modal?

Uma janela de diálogo (pop-up) que aparece sobre o conteúdo da página.

Propósito

Usado para notificações, formulários (login), confirmações ("Tem certeza?") ou para mostrar conteúdo extra.

Como funciona?

É acionado por um botão (data-bs-toggle="modal") e requer o JavaScript do Bootstrap para funcionar.

Anatomia do Modal

Um modal é composto por 3 partes principais.

Header (.modal-header)

Contém o título (.modal-title) e o botão de fechar (.btn-close).

Body (.modal-body)

O conteúdo principal do seu modal (texto, formulários, imagens).

Footer (.modal-footer)

Onde ficam os botões de ação (Salvar, Fechar, Confirmar).

Boas Práticas com Modais

Não Abuse

Modais interrompem o fluxo do usuário. Use com moderação e apenas para informações essenciais.

Seja Focado

Use modais para tarefas curtas e diretas, como um formulário de login ou uma confirmação de exclusão.

Tenha Saídas Claras

Sempre inclua um botão de fechar (`.btn-close`) E um botão de "Fechar" ou "Cancelar" no rodapé.

Resumo: Modal

Acionador

Use data-bs-toggle="modal" e data-bs-target="#ID" em um botão.

Estrutura

.modal > .modal-dialog > .modal-content > (header, body, footer).

JavaScript

O JavaScript Bundle do Bootstrap (bootstrap.bundle.min.js) é obrigatório para funcionar.

Backdrop Estático

Tente clicar fora deste modal. Ele não vai fechar. Use o atributo data-bs-backdrop="static".

Código: Modal Básico
<!-- 1. O Botão Acionador -->
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#meuModal">
  Abrir Modal
</button>

<!-- 2. A Estrutura do Modal (pode ir em qualquer lugar) -->
<div class="modal fade" id="meuModal" tabindex="-1">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">Título do Modal</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">
        <p>Este é o conteúdo do modal.</p>
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Fechar</button>
        <button type="button" class="btn btn-primary">Salvar</button>
      </div>
    </div>
  </div>
</div>

<div class="modal-dialog modal-dialog-scrollable">
  <div class="modal-content">
    <div class="modal-header">...</div>
    <div class="modal-body">
      (Muito conteúdo aqui...)
    </div>
    <div class="modal-footer">...</div>
  </div>
</div>

  <div class="modal-dialog">
    ... (Não fecha ao clicar fora)
  </div>
</div>
